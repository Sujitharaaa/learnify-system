# Learnify - Instructor Dashboard

A web-based instructor dashboard system built with React, Vite, and Tailwind CSS as part of a university assignment.

## About

Learnify is a smart interactive learning system that empowers instructors to manage courses, assignments, quizzes, and monitor student performance analytics.

## Features

- Instructor Dashboard with real-time course statistics
- Manage Courses (add, edit, view, delete)
- Manage Assignments with submission tracking
- Quiz management and grading
- AI Instructor Summary with performance insights

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Setup

Run `npm install` to install dependencies.

Run `npm run dev` to start the development server.
